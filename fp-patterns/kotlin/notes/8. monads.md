# The Famous Monad

## Monads are a pattern for sequencing computations
```
  1. do x
  2. do y, that uses x
  3. do z, that uses y
  4. return z`
```
* A monad interface (pseudocode):
```
  interface Monad[A] {
    fun bind[B](f: A => Monad[B]): Monad[B]

    companion object {
      fun unit[A](x: A): Monad[A]`
    }
  }
```
* Monad laws
  * Left-identity:
      `unit(x).bind(f) == f(x)`
  * Right-identity law:
      `m.bind(unit) == m`
  * Associativity:
      `m.bind(f).bind(g) == m.bind(x â‡’ f(x).bind(g))`

* Instances can be written for List, Option
